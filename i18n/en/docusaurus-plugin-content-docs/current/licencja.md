OpenKSeF operates on tax data (invoices from KSeF). Before production deployment, we recommend:

1. Security configuration review by a qualified person
2. Implementing HTTPS, backups, and monitoring
3. Compliance with data protection regulations (GDPR)
4. Regular updates to the latest versions
